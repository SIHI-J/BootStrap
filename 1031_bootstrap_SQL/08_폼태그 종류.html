부트스트랩 08_폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 컴포넌트의 예시와 사용 가이드입니다.

폼태그 사용시 form, fieldset, legend태그 사용.

폼 양식 제목(ex. 로그인, 회원가입)

(패스워드는 특수문자를 포함 8-10까지 가능)

class="form-text"는 작성시 설명문구를 넣고자 할 경우 사용

폼 입력 양식의 크기 조정은 .form_control_lg, .form-control-sm으로 크기조정이 가능

.form-text를 사용하면 block요소와 간격을 쉽게 맞출 수 있도록 상단 여백이 추가. small, span을 사용하면 좋음.

버튼 비활성화 - disabled

disabled 불리언 속성을 추가하면 비활성화 외형이 되며, 포인터 이벤트가 삭제되면서 포커싱되지 않습니다.

읽기 전용 -readonly

readonly 불리언 속성을 추가함으로써 입력값의 변경을 막을 수 있습니다 disable과 달리 readonly입력 값은 계속 포커싱 및 선택이 가능합니다.

라디오 버튼 - label태그에 .form-check-label, input태그에 .form-check-input을 적용

체크박스 - label태그에 .form-check-label, input태그에 .form-check-input을 적용

선택 목록 - select option

class="form-select"를 적용시 가로 100%로 커짐. 크기조정시에는 .form-select-lg, .for-select-sm을 적용

select태그에 multiple속성을 넣으면 다중 선택도 가능. size속성으로 크기도 조정가능.

다중 텍스트 박스

.form-control일때는 가로 100%, form-control-lg, form-control-sm을 적용할 수 있음.

높이를 적용하고 싶으면 css 서식으로 조절가능

파일 첨부

.input-group안에서 input, label태그의 순서를 변경하면 'upload'제목의 앞, 뒤 순서가 변경됨.
