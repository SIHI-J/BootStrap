표(table) 관련 서식

  • class="table" : 테이블 기본서식이 적용됨.
  • class="table table-striped" : 각 행마다 홀수/짝수번째 색상이 다르게 적용.
  • class="table table-bordered" : 테이블 태그에 테두리 적용
  • class="table table-hover" : 테이블에 마우스 오버시 색상전환
  • thead class="table-dark" : 셀 배경색 어둡게
  • thead class="text-center" : 가운데 정렬
  • class="table-success" : 셀 배경색 초록색으로
  • class="table-danger" : 셀 배경색 빨간색으로
  • class="table-info" : 셀 배경색 시안색으로
  • class="table-primary" : 셀 배경색 파란색으로
  • class="table-responsive" : 화면 너비가 768px미만일 경우 가로로 스크롤 됨

    • 셀 배경 색상 클래스는 tr, td 각각 분류 색상을 적용할 수 있음
