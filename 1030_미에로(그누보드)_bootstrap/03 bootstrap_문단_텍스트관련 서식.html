2. 글자 관련

small : 글자를 작게 설정(14px)

mark : 글자에 하이라이트 효과를 적용(16px)

abbr : 약어나 줄임말을 사용하고자 할 때 사용.(16px)

인용문
: 다른 사람의 말이나 글을 대신하여 작성하는 글(글자크기 : 14px, 색상 : #6c757d)

q : 인용문은 양쪽 ""기호가 표시됨.

dl, dt - 용어를 정의하고자 할때 사용하는 태그
dd - 용어에 대한 설명을 적기 위한 태그
code : 문서안에서 태그이름을 표시하기 위한 태그로서 색상이 '자주색'으로 변경됨. //1.변수선언 let gnb = document.getElementById('#gnb');

글자의 중요도에 따라 다르게 색상 적용하기(클래스명)

  • text-muted : (회색) 텍스트의 중요도에 따라 문자의 색상을 구분하기
  • text-primary : (하늘색) 텍스트의 중요도에 따라 문자의 색상을 구분하기
  • text-success : (초록색) 텍스트의 중요도에 따라 문자의 색상을 구분하기
  • text-info : (시안색) 텍스트의 중요도에 따라 문자의 색상을 구분하기
  • text-warning : (노란색) 텍스트의 중요도에 따라 문자의 색상을 구분하기
  • text-danger : (빨간색) 텍스트의 중요도에 따라 문자의 색상을 구분하기

- 상황별로 문자의 배경색을 적용하려면 class='bg_primary', class='bg_success', class='bg_info', class='bg_warning', class='bg_danger'을 p요소 안에 작성한다.

문단 관련 서식

class= "lead" : 단락을 눈에 띄게 하기(글자크기 : 20px)

class= "small" : 상위 문자 크기의 85% = 14px로 문자 크기를 작게 설정함.

class= "text-start" : 문단정렬(왼쪽정렬)

class= "text-center" : 문단정렬(가운데 정렬)

class= "text-end" : 문단정렬(오른쪽정렬)

class= "text-justify" : 문단정렬(양쪽배분정렬)

class= "text-uppercase" : 영문자를 무조건 대문자로 출력

class= "text-Lowercase" : 영문자를 무조건 소문자로 출력

class= "text-capitalize" : 영문 맨앞글자만 무조건 소문자로 출력
